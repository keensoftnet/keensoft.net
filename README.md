# keensoft.net

The official Keen Software company website — built with [Next.js](https://nextjs.org), TypeScript, and Tailwind CSS.

## Pages

| Route | Description |
|---|---|
| `/` | Home — hero, features, products overview, testimonials, CTA |
| `/about` | About — mission, values, timeline, team |
| `/products` | Products — detailed product pages and pricing |
| `/contact` | Contact — enquiry form, contact details, FAQ |

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
